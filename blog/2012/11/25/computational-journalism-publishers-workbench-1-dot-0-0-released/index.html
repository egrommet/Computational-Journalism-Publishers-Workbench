Computational Journalism Publishers Workbench 1.0.0 Released

Nov 25th, 2012 | Comments

Quick Start

  • Quick Start - Generic Linux on VMware Player
  • Quick Start - Fedora 17 / 18 Beta on VirtualBox
  • Quick Start - Linux Mint 14 on VirtualBox

What’s New?

  1. Now builds on the five most popular Linux distributions
    • Linux Mint 13+
    • Ubuntu 12.04+
    • Mageia 2+
    • Fedora 17+
    • openSUSE 12.2+
  2. Now works with all major Linux desktops
    • GNOME
    • KDE
    • XFCE
    • LXDE
    • MATE
    • Cinnamon
    • Unity
  3. Runs on bare metal or as a guest virtual machine with VirtualBox or Virtual Machine Manager / qemu-kvm hosts
  4. New applications:
    • Redis 2.6 with Perl, Ruby and R client libraries
    • Perl, Ruby and R Twitter API libraries
    • Shiny R application framework
    • cranvas R exploratory data analysis framework
    • Linux profiling tools and Pretty Damn Quick (PDQ) Queueing Model Package
