Computational Journalism Publishers Workbench 1.1.0 Released

Dec 12th, 2012 | Comments

Quick Start

  • Quick Start - Generic Linux on VMware Player
  • Quick Start - Fedora 17 / 18 Beta on VirtualBox
  • Quick Start - Linux Mint 14 on VirtualBox

What’s New?

The are numerous minor tweaks, bug fixes, etc., but the major new feature is that VMware Player 5 and VMware Workstation 9 are now supported hosts. Fedora 18 Beta is a supported Linux distribution as well, and is my preferred operating environment.

Road Map

The next major update will feature Windows 8 Pro ‘Client Hyper-V’ hosting. I have this working with my laptop when connected to both wired and wireless networks, but there are some configuration issues with wireless networking alone. The other major issue with the Windows 8 Pro ‘Client Hyper-V’ hosting is that you can’t have both Hyper-V and VMware Workstation active at the same time. I’m tentatively planning to release this on December 19, 2012.

Other coming attractions:

  1. The Shiny project is very active. There is a server version in the works and I’m tracking that in more or less real time.
  2. I am considering selling pre-built appliances with the workbench software. The big stumbling block is branding; I have to remove all traces of the upstream Linux distribution’s branding and create my own. I think it’s worth doing, though - it takes a long time to build one of these appliances from the ground up. Most likely these will be built with SUSE Studio and be based on openSUSE Linux 12.2 32-bit version.
