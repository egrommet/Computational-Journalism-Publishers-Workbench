CompJournWin8Pro 0.0.7 Released

Apr 5th, 2013 | Comments

After some intense thrashing around, installing/uninstalling/rebooting, etc., I’ve pushed another release. I’m calling this one v0.0.7, mostly because seven is a lucky number. Special thanks to Tal Galili for the awesome tool that makes this possible, the ’installr’ package.

What’s In It?

  • R
  • GGobi
  • RStudio
  • Rtools
  • MikTeX
  • git
  • GGobi
  • R Commander
  • Rattle
  • Graphics and Reproducible Research task views
  • A generous collection of useful CRAN packages

Road Map

  1. v0.0.8 - Editing option
    • Pandoc will be added to the Base.
    • Calibre
    • Sigil
    • LyX
    • GIMP
    • Avidemux
    • Blender
    • Audacity
    • CRAN Audio packages
  2. v0.0.9 - PostgreSQL option
  3. v0.0.10 - Perl option - Strawberry Perl + Padre
  4. v0.1.0 - Full platform release
