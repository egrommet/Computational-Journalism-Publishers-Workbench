Computational Journalism Publishers Workbench Version 0.1.0 Released

Aug 6th, 2012 | Comments

For a variety of reasons, I’ve stopped working on the SUSE Studio versions of my tool collections. I’ve discontinued distribution of Data Journalism Developer Studio 2012LX. I’m still distributing the Computational Journalism Server, but all enhancements are on hold until openSUSE 12.2 is released and the repositories are available in SUSE Studio. There has not yet been a commitment from the openSUSE project on a date for that release.

In the interim, I’m focusing on the desktop use cases with the Computational Journalism Publishers Workbench. I’ve taken the scripts I’ve been using to install the tools for previous appliances and ported them to three Linux desktops – Fedora 17 LXDE, Lubuntu 12.04 and openSUSE 12.2 Release Candidate 2. I am planning to port them to Linux Mint Debian Edition, Linux Mint 13 and Mageia Version 2 in the next few weeks.

The rough road map is to release 0.x.0 every Monday for the next five weeks, then revisit some of the major use cases. In particular, I’m planning to enhance the PDF data extraction functionality and the Twitter research tools. The main over-arching goal is to build a platform for the independent publishing equivalent of reproducible research. In my opinion the whole publishing tool set – research, reporting, editing and publishing – needs to be reproducible and as automatic as possible. The core tools are similar to what you’ve seen from the previous collections – Linux and R Studio.
